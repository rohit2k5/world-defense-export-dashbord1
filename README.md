# 🌍 World Defense Exports Dashboard

## 📊 Project Overview
The **World Defense Exports Dashboard** provides an interactive analysis of global defense exports.  
It helps users explore exporting countries, trends over time, and distribution of exports across nations.

---

## 🚀 Key Features

- 📈 **Total Exports Overview**
  - Displays total exports value: **$1.83 Trillion**
  - Shows overall percentage distribution

- 🌎 **Top Exporting Countries**
  - United States (Top exporter)
  - European Union
  - Russian Federation

- 📅 **Time Series Analysis**
  - Trend of exports over years
  - Identifies increase/decrease patterns

- 🗺️ **Geographical Visualization**
  - World map showing exporting countries

- 📊 **Data Visualizations**
  - Bar Chart → Top exporters
  - Pie Chart → Export share by country
  - Treemap → Country-wise comparison
  - Line Chart → Export trends over time

- 🔍 **Filters / Slicers**
  - Year filter
  - Country selection

## 📂 Dataset Details

### Exporter Countries Table
| Column Name     | Description |
|----------------|------------|
| Country Code    | Unique country identifier |
| Country Name    | Name of the country |
| Exports         | Export value |
| Percent         | Contribution percentage |
| Total Country   | Count of countries |
| Total Exports   | Total export value |
| Years           | Year of record |

### Importer Countries Table
| Column Name     | Description |
|----------------|------------|
| Attribute       | Category |
| Country Code    | Unique identifier |
| Country Name    | Name of importing country |
| Value           | Import value |
